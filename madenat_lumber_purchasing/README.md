# MADENAT Lumber Purchasing

Gestión especializada de compras de madera para MADENAT.

## Propósito

Órdenes de compra automáticas para madera, con tracking de volumen recibido vs comprado, y consola de compras con KPIs.

## Dependencias

- `madenat_lumber_core`
- `purchase`

## Modelos principales

- `purchase.order` (extendido) — Campos de volumen, tracking de recepción
- Vistas especializadas para madera (list/kanban con KPIs)
