# MADENAT Vendor Payment Management

## Descripción
Sistema profesional de gestión de pagos a proveedores con trazabilidad completa para MADENAT.

## Características
- Extensión de costos de embarque con trazabilidad completa
- Registro de proveedores por tipo de servicio
- Vinculación con facturas y pagos contables
- Gestión documental de respaldos
- Estados de pago (pendiente, parcial, pagado)
- Órdenes de pago con aprobación multinivel

## Instalación
1. Copiar el módulo a `/opt/odoo18/custom_addons/`
2. Actualizar lista de módulos
3. Instalar desde Apps

## Dependencias
- base
- account
- madenat_lumber_logistics
